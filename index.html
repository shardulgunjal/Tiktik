🔔 Tiktik

Dynamic Island–style notifications for the modern web. Silky GSAP animations, accessible, and endlessly customizable.

v2.0.0 • TypeScript • GSAP Powered • WCAG Accessible

🎨 Toast Types

Each type has its own gradient, icon, and accent color.

📊 Progress Bar

Visual countdown showing time remaining before auto-dismiss.

📍 Positions

Show toasts in any corner or edge of the screen.

📚 Deck Stacking

Toasts overlap in a card-deck. Hover the stack to expand. Toggle between modes.

👈 Swipe to Dismiss

Drag any toast left or right to swipe it away. Spring-back if you don't drag far enough.

⏳ Promise Toast

Automatically transitions from loading → success or error.

👆 Interactivity

Expandable content on hover, action buttons, and click callbacks.

🖼️ Custom HTML

Inject any HTML content into the toast body.

⏳ Manual Loading

Persistent loading toast, dismiss it manually.

🧩 Headless Mode

Provide your own DOM via a custom render function. Tiktik handles lifecycle, timers, swipe, and animations.

⌨️ Focus Management

Press Alt+T to jump focus to the newest toast. Toasts with action buttons auto-focus and trap Tab.

💻 Usage

Get started in seconds — import and call.

// npm install tiktik gsap
import Tiktik from 'tiktik';

// Simple toast
Tiktik.success('Saved!');

// With options
Tiktik.showToast({
  message: 'Hello World',
  type: 'info',
  duration: 5000,
  position: 'bottom-right',
  progress: true,
});

// Promise toast
Tiktik.promise(fetch('/api/data'), {
  loading: 'Fetching…',
  success: 'Data loaded!',
  error: 'Failed to load',
});
